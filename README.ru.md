# Казуальный прототип "Blast" на Cocos Creator

[English](./README.md)

Прототип казуальной головоломки с механикой Blast, разработанный в рамках технического задания для компании **TapClap**.

* **Играть в браузере (GitHub Pages):** https://Ne0Git.github.io/cocos-blast-prototype (Поддерживает Desktop и Mobile браузеры).

## 🛠️ Технологический стек
* **Движок:** Cocos Creator 2.4.11
* **Язык:** TypeScript
* **Тестирование:** Jest / ts-jest

## 📐 Архитектурные особенности
1. **Разделение Логики и Отображения:** Все математические расчеты (алгоритм поиска групп Flood Fill, матричное схлопывание столбцов, начисление очков) написаны на чистом TypeScript. Логика полностью изолирована от движка Cocos Creator, что позволило покрыть её автоматизированными юнит-тестами.
2. **Оптимизация памяти (Object Pool):** Для предотвращения пиков сборщика мусора (Garbage Collector) и микро-фризов на мобильных устройствах используется `cc.NodePool`.
3. **Система настройки уровней (Data-Driven)):** Параметры уровней вынесены в конфигурационные JSON-файлы. Настройки размеров поля, доступных цветов и целей победы подгружаются динамически через `LevelManager`. Реализован бесконечный режим с рандомизацией при завершении прописанных уровней.
4. **Масштабируемая система комбо:** Настройки создания бустеров вынесены в декларативный массив в Редакторе. Добавление новых бустеров происходит без изменения логики обработки хода.

## 🕹️ Реализованные механики
* **Основные механики:** Сжигание групп от 2-х блоков, падение кубиков сверху, спавн новых, подсчет ходов/очков, экраны победы и поражения с цикличным перезапуском.
* **Бустер "Бомба":** Активируется по кнопке из инвентаря. Уничтожение блоков в радиусе R (настраивается в редакторе).
* **Бустер "Телепорт":** Активируется по кнопке из инвентаря. Свап двух любых блоков на поле.
* **Бустер "Ракета":** Спавнится на поле. Автоматически определяет направление (горизонтальная/вертикальная) на основе геометрии взорванной группы. Уничтожает ряд или столбец.
* **Бустер "Супер-бомба":** Спавнится на поле. Уничтожение блоков в радиусе R (подразумевается больший радиус, настраивается в редакторе).
* **Автоматическое перемешивание поля:** Если нет возможности сжечь тайлы перемешивает блоки на поле до 3 раз (настраивается в конфиге для каждого уровня), далее проигрыш.

## 💻 Как запустить проект локально

> [!NOTE]
> Для запуска и редактирования проекта вам понадобится установленный **Cocos Creator версии 2.4.11**.

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/Ne0Git/cocos-blast-prototype
   ```
2. **Откройте проект:**
   * Запустите Cocos Dashboard.
   * Нажмите **Add** (Добавить проект) и выберите папку с клонированным репозиторием.
3. **Запуск:**
   * Откройте сцену `assets/scenes/Main.fire` в редакторе.
   * Нажмите кнопку **Play** в верхней панели для запуска в браузере.

## 🛠️ Как запустить тесты локально
1. Установите зависимости: `npm install`
2. Запустите Jest-тесты: `npm test`
